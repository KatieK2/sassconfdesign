Styleguide

This is the design team style guide for Sass Conf. The design is open source to fork and contribute back but it is not licenced under an open source licence. Please do not steal this design for any use other than Sass Conf.

This logo is an extenstion of the new Sass logo with numerous stylistic tweaks. There is both a large and a small version, the large version should be no smaller than 200px. To use it, download the font file. You can write - for the full logo and S for the smaller logo. Also included are Facebook and Twitter icons. All lowercase characters have no width and result you can write anything for screen readers and SEO.

Color

  • $primary
  • $secondary
  • $tertiary
  • $quadrary

This is an analogous palette derrived from a slight variation of the primary Sass color. The scheme was built with the color-schemer Compass plugin to be easily adjusted. We may end up using the secondary, tertiary, and quadrary colors more than the primary and that is ok, it will make primary pop more.

Usage of black is not allowed on this site. If you need a dark color, use $quadrary but never use black because it sucks the color out of designs.

When using these colors in the building of the website always use the variables, never ever use the hex values. That way we can quickly adjust the overall site colors without worrying about where they are used.
